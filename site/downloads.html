Downloads

To get the latest development release of Apache Felix, you can check out the sources and build Felix yourself. Otherwise, the releases below are available for download. To install, just download and extract. These KEYS can be used to verify the release archive.

All Apache Felix products are distributed under the terms of The Apache Software License (version 2.0). See our license page, or the LICENSE file included in each distribution.
