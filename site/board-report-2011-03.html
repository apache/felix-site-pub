Licensing and other issues

  • Discussed the proper way to handle version numbers for canceled release votes (e.g., canceling a release vote to fix an issue). The community was split between allowing the version number to be reused on the subsequent release versus disallowing it. Since there was no consensus, we decided to not enforce a policy and simply let the person performing the release decide.
