bundle:remote-clean

Full name:

org.apache.felix:maven-bundle-plugin:3.5.0:remote-clean

Description:

Clean a remote repository file. It just looks for every resources and check that pointed file exists.

Attributes:

  • Binds by default to the lifecycle phase: clean.
